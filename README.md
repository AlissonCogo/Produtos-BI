Trazendo uma divisão clara sobre Produtos, Sub-Produtos, Mes/Ano.

- Contagem de Cliente
- Faturamento (Por Produto)
- Faturamento (Por Sub-Produtos)
- PorcentagemPorChave
- Total Faturamento
- Total Vendas 2019
- Total Vendas 2020
